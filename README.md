# vinmonopolet-cli

CLI to search for products on Vinmonopolet.

![Image](https://www.dropbox.com/s/25oybuukf79i59h/vmp_cli.png?dl=1)

## Installation
```bash
$ npm install -g vinmonopolet-cli
```

## Usage (command line)

```bash
$ vmp --help // List commands
```

```bash
$ vmp -s <title> // wildcards supported
```
